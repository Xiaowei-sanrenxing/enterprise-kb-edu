# 教培行业 黑话词典 · 让 agent 看懂内部术语

> 行业通用术语已预填。公司专属术语 / 人物 / 产品请客户补在文末。

## 业务术语（行业通用）
- **线索**：尚未成交的潜在客户（家长/学员/决策人）。
- **私域**：可反复触达的自有流量池（微信/社群），区别于公域平台流量。
- **提纯**：把泛流量筛成精准目标客户的过程（vs 单纯放量）。
- **钩子/诱饵**：吸引目标客户留资/加微的内容或工具。
- **承接**：流量进来后的接待动作（自动回复/人工）。
- **续班/续费**：学员在课程到期后继续报名下一阶段。
- **老带新/转介绍**：老学员推荐新客户。
- **陪跑**：交付期对学员的持续跟进辅导。

## 本体系关键概念（跨工作流复用）
- **工作流**：一件能整个交给 agent 干完的事（获客/销转/研发/交付/续费），是本库组织主轴。
- **任务卡 6 要素**：目标/上下文/指令/工具/权限/验收，每条工作流一张。
- **客户分级 A-F**：销转第一步，30 秒判级决定打法（见销转工作流）。
- **三层漏斗**：获客的顶层放量/中层提纯/底层成交（见获客工作流）。
- **钩子精准度公式**：精准度 ≈ 摩擦门槛 × 行业专属性。
- **带走物**：每讲学员能带走的工具/模板，没有=听完就忘（见研发工作流）。
- **价值节点**：学员体验到价值的时刻，是续费/转介绍最佳窗口（见续费工作流）。
- **双区**：草稿区/定版区/归档区，定版才可对外。
- **受限/脱敏**：学员隐私受限，对外引用必脱敏。
