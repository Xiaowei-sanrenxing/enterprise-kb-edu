# 任务卡 · 教学交付（teaching-delivery）

> Agent 操作总台。接到"排课/上课/作业批改/答疑/学员陪跑/交付复盘"类任务，从这里开始。
> ⚠️ **本工作流涉及学员隐私，全程受限。动手前必读权限铁律。**

## 🎯 目标(Goal)
把已报名学员，从开课带到学有所得/完成交付，全程做好陪跑与答疑，**并把可复用的交付方法论脱敏沉淀**。

## 📥 上下文(Context) —— 动手前必读全
1. [`context/教学交付流程.md`](context/教学交付流程.md) — 建档→排课→上课→作业答疑→陪跑→复盘
2. [`context/受限与脱敏铁律.md`](context/受限与脱敏铁律.md) — ★本工作流最高优先级，违反=隐私事故
3. 〔学员档案〕 — 客户补：🔒 受限，存 `_受限/`
4. 〔答疑知识库〕 — 客户补：高频问题与标准答复

## 🪜 指令(Instructions) —— 标准动作流
1. **建档**：学员档案进 `_受限/`，标 `permission: 受限`
2. **排课/上课**：按议程推进，记录出勤/进度
3. **作业答疑**：批改 + 答疑，高频问题登记 → 回流课程研发的"上期问题表"
4. **陪跑**：按节点主动跟进学员，记录陪跑情况（仍受限）
5. **复盘沉淀**：可复用的交付方法论 → **必须脱敏**（去真名改化名）后才能进 `定版/`
6. 任何对外案例/物料引用学员数据 → 先脱敏

## 🛠 工具(Tools)
〔学员档案系统〕 · 〔答疑知识库检索〕 · 作业批改模板 · 陪跑节点提醒

## 🔒 权限(Permission)
- 🔒 **本工作流默认全部受限**：学员档案/作业/答疑记录/陪跑记录 = 隐私
- 🔴 受限内容：永不进对外 `定版/`、永不进飞书发布
- ✅ 仅例外：**已脱敏**的可复用交付方法论可进 `定版/`
- 需人工确认：任何含学员信息的对外产出

## ✅ 验收(Acceptance)
- 学员数据全在 `_受限/` ｜ 对外产出无真名/手机/微信 ｜ 高频答疑有回流研发
- 详见 evals.md

## 🔄 自我进化接口
答疑高频问题 + 陪跑反馈 → 脱敏后回流 course-development 上期问题表 → 下期课程修掉。

# 受限与脱敏铁律（★本工作流最高优先级）

> 违反 = 隐私事故。教学交付涉及学员隐私，这是不可妥协的红线。

## 三条铁律
1. **本板块默认全部受限**：学员档案/作业/答疑/陪跑记录 `permission: 受限`，永不进对外 `定版/`、永不进飞书。
2. **定版门槛 = 脱敏**：未脱敏一律留 `_受限/`；只有脱敏后的可复用方法论能升 `定版/`。
3. **对外引用必脱敏**：做招生案例/对外物料引用学员故事 → 去真名（改化名）、去手机/微信、去任何可定位个人的信息。

## 脱敏检查清单（升定版前逐项过）
- [ ] 无真实姓名（已改化名）
- [ ] 无手机号/微信号/身份证
- [ ] 无可定位到个人的细节组合
- [ ] 无未授权的第三方信息

## agent 行为
不确定能不能对外 → **默认按受限处理，问负责人**。宁可保守，不可泄漏。

# 权限分级 · 四级

> 企业库和个人库最硬的区别。没有权限治理 = AI 资料垃圾场 + 隐私事故。

| 级别 | 含义 | 典型资产 | agent 行为 |
|---|---|---|---|
| 🟢 公开 | 可对外发布 | 对外物料、方法论、介绍 | 可写入对外产物 |
| 🟡 内部 | 团队内用 | 话术、SOP、运营数据 | 内部可用,不对外 |
| 🔴 受限 | 隐私/敏感 | 客户对话、通话记录、手机号、名单 | **绝不写入对外产物,只内部分析** |
| ⛔ 禁止入库 | 不该进库 | 证件、银行信息、未授权第三方资料 | 拒绝入库 |

## agent 红线
1. 🔴 受限资产 → 输出时必须脱敏/打码,绝不原文外发
2. 超标准的报价/优惠/对外承诺 → 必须人工确认
3. 不确定权限 → 默认按更高一级处理,问用户

# 教培行业 · 合规与权限默认值

> 合成进成品库 `90-governance/行业合规默认.md`。客户可按自身情况收紧。
> ⚠️ 教培是强监管行业，对外宣传/收费/排课前必读本页。各地细则可能更严，以当地教育/市监要求为准。

## 默认权限分级（通用四级 + 教培补充）
| 资产类型 | 默认级别 |
|---|---|
| 学员/家长个人信息（姓名/手机/微信/成绩/档案）| 🔴 受限（永不对外、输出必脱敏改化名）|
| 未成年人相关信息 | 🔴 受限（更严，去任何可定位个人的细节）|
| 内部话术/SOP/课件/运营数据 | 🟡 内部 |
| 对外招生物料/科普/品牌介绍 | 🟢 公开（须经合规审核，见下方红线）|
| 办学许可/教师资质证照 | ⛔ 禁止入库（证件类，留原地索引）|

## 教培行业红线（agent 必须遵守，对外产出前逐条过）
1. **禁虚假宣传**：不得承诺/暗示"保过、保升学、保提分 X 名"，不得虚构或夸大升学率、师资、获奖、效果。广告禁绝对化用语（最好/第一/100%）。
2. **收费合规**：学科类一次性收费跨度不得超过 **3 个月或 60 课时**（取其短），不得超期预收；遵守预付费资金监管要求。
3. **双减约束（学科类）**：不得占用国家法定节假日、休息日及寒暑假组织学科类培训；排课/招生文案不得违反。
4. **教学合规**：不得超前超纲教学；课程承诺与实际一致。
5. **资质公示**：学科类教师须具备相应教师资格并依规公示；对外不得虚标师资。
6. **未成年人保护**：涉及未成年学员的信息与肖像，未经监护人授权不得用于对外宣传。

## agent 行为
- 任何对外招生/宣传产出（发公众号、给渠道、写海报文案）→ **先过上面 6 条红线**，命中任一 → 改写或人工确认。
- 涉及收费周期、升学承诺、节假日排课 → **必须人工确认**，agent 不得自行拍板。
- 不确定是否合规 → 默认按更严处理，问负责人。

> 合规是教培的生命线。一条虚假宣传可能导致整个机构被处罚——对外内容宁可保守。
