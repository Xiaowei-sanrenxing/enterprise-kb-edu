# 续费节点打法

> 核心：**在学员"体验到价值"的节点推续费，不是临到期才催。**

## 为什么不靠催
到期日催续费 = 把决策压在"还想不想花钱"上，抗拒最强。价值节点推 = 把决策建在"我已经получ得到了"上，顺势而为。

## 价值节点（按这些时机推，客户填自己课程的具体节点）
| 节点 | 信号 | 动作 |
|---|---|---|
| 首次出成果 | 学员做出第一个可见效果 | 肯定成果 → 自然引出进阶 |
| 阶段达成 | 完成一个里程碑 | 复盘收获 → 推下一阶段 |
| 高满意时刻 | 主动表达感谢/认可 | 满意峰值 → 续费 + 转介绍双触发 |
| 临到期 | 课程将结束 | 兜底动作，但不应是唯一动作 |

## 用法
盯学员的**价值节点**而非日历。节点到了，用成果说话推续费/进阶。
