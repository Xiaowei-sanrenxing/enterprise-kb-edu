# 受限与脱敏铁律（★本工作流最高优先级）

> 违反 = 隐私事故。教学交付涉及学员隐私，这是不可妥协的红线。

## 三条铁律
1. **本板块默认全部受限**：学员档案/作业/答疑/陪跑记录 `permission: 受限`，永不进对外 `定版/`、永不进飞书。
2. **定版门槛 = 脱敏**：未脱敏一律留 `_受限/`；只有脱敏后的可复用方法论能升 `定版/`。
3. **对外引用必脱敏**：做招生案例/对外物料引用学员故事 → 去真名（改化名）、去手机/微信、去任何可定位个人的信息。

## 脱敏检查清单（升定版前逐项过）
- [ ] 无真实姓名（已改化名）
- [ ] 无手机号/微信号/身份证
- [ ] 无可定位到个人的细节组合
- [ ] 无未授权的第三方信息

## agent 行为
不确定能不能对外 → **默认按受限处理，问负责人**。宁可保守，不可泄漏。
